Getting Started

Create a Vecta Application in 2 simple steps:

  1. Create a new VectaConfig object.

    The VectaConfig object contains information about the application you want to create.

    Fields Type Default Usage
    Title String "Vecta Engine" The title of the application that is shown on the application window.
    Width int 1280 The width of the application window.
    Height int 720 The height of the application window.
    Version String "0.0.1" The current version of the application.

  2. Create a new class that extends the VectaApplication class. VectaApplication takes a VectaConfig as an argument.

    The VectaApplication class contains the main game loop.
